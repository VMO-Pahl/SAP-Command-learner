```
6. Save
```

## 🐛 Häufige Anfängerfehler

### ❌ Fehler 1: Falsche Branch
```
→ Sicherstellen: Branch ist "main" (nicht "master")
```

### ❌ Fehler 2: Datei liegt nicht im Root
```
→ index.html muss ganz oben sein, nicht in Ordner!
```

### ❌ Fehler 3: Repository ist Private
```
→ GitHub Pages braucht Public (außer mit Pro)
```

### ❌ Fehler 4: 404 Error sofort nach Upload
```
→ Normal! GitHub Pages braucht 1-2 Minuten
→ Einfach kurz warten und neu laden
```

### ❌ Fehler 5: App lädt aber zeigt keine Befehle
```
→ Browser-Console prüfen (F12)
→ JavaScript muss enabled sein
→ Internetverbindung prüfen (für Claude API)
```

# ⚡ Quick Start (5 Minuten)

## Die 3 Schritte zum Erfolg

### 1️⃣ Repository erstellen
```bash
git clone https://github.com/[dein-username]/sap-command-learner.git
cd sap-command-learner
```

Oder manuell auf GitHub:
- Neues Repository: https://github.com/new
- Name: `sap-command-learner`
- Public wählen
- Erstellen

### 2️⃣ Dateien hochladen
Folgende Dateien in dein Repository:
- ✅ `index.html` ← Die App selbst
- ✅ `README.md` ← Dokumentation
- ✅ `.gitignore`

**Einfach**: Auf GitHub - "Add file" - "Upload files"

### 3️⃣ GitHub Pages einschalten
1. Repository → Settings
2. Pages
3. Branch: `main` / Folder: `/ (root)`
4. Save

**Fertig!** Deine App läuft unter:
```
https://[dein-username].github.io/sap-command-learner/
```
